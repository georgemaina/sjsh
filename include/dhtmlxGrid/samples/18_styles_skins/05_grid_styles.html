Setting Grid styles

View page source

There are two ways to set grid styles:

  • using modified style sheet
  • changing parameters on the fly using script methods. In dhtmlxGrid you are allowed to use setCellTextStyle, setRowTextStyle and setStyle methods.

    • Styles:
  • Default Style
  • M1 Style
  • M2 Style
    		•

      Parameters passed to the constructor for setCellTextStyle method are the following:
    • row_id - row id
    • ind - cell index
    • styleString - style string in common format (exmpl: "color:red;border:1px solid gray;")
      Parameters you should specify in setRowTextStyle method are:
    • row_id - row id
    • styleString - style string in common format (exmpl: "color:red;border:1px solid gray;")
      Parameters for the last method are the following:
    • ss_header - style def. expression for header
    • ss_grid - style def. expression for grid cells
    • ss_selCell - style def. expression for selected cell
    • ss_selRow - style def. expression for selected Row
